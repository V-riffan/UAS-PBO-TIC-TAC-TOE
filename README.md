Deskripsi Singkat

Proyek ini adalah permainan Tic-Tac-Toe yang dikembangkan menggunakan framework Greenfoot sebagai tugas mata kuliah Pemrograman Berorientasi Objek (PBO).
Game berjalan pada papan 3×3 dan mendukung dua mode permainan: Player vs Player serta Player vs Bot.

Tujuan
* Mengimplementasikan konsep dasar OOP (class, objek, inheritance, dan komposisi).
* Membuat game interaktif berbasis event (klik) dengan Greenfoot.
* Menerapkan logika permainan, algoritma pemeriksaan kemenangan, serta kontrol game state.
* Menyediakan pengalaman bermain sederhana dengan antarmuka visual.

Cara main
1. Buka Greenfoot.
2. Import file UASPBO.gfar ke dalam Greenfoot.
3. Tekan tombol Run.
4. Pada tampilan awal, klik StartButton untuk masuk ke menu.
5. Pilih mode permainan:
6. Play 1 vs 1
7. Play vs Bot
8. Klik pada Cell (kotak kosong) untuk bermain.

Cara kerja
1. Pemain giliran pertama menggunakan simbol X.
2. Jika mode bot dipilih, komputer akan menggunakan simbol O.
3. Klik sel kosong untuk menempatkan tanda.
4. Pemenang ditentukan bila salah satu pemain berhasil menyusun tiga simbol berurutan secara horizontal, vertikal, atau diagonal.
5. Jika tidak ada pemenang dan semua sel penuh → pertandingan seri.
6. Gunakan ResetButton untuk memulai permainan baru tanpa kembali ke menu awal.
